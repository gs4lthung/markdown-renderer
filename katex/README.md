# KaTeX Library

This directory contains the KaTeX library for math rendering.

## Files Included

- katex.min.css - KaTeX stylesheet
- katex.min.js - KaTeX JavaScript library

## Usage

The KaTeX library is loaded locally to avoid CSP violations in Chrome extensions.

## Font Files

KaTeX requires font files for proper rendering. For now, we'll use CDN links in the CSS or web fonts. If needed, download fonts from: https://github.com/KaTeX/KaTeX/tree/master/dist/fonts
